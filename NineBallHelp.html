About the 9-Ball Scoreboard

  • Tracks state of the current rack as well as innings, defensives, time-outs, and player points earned/needed to win for a match
  • Keeps record of 9-snaps, 9-and-runs
  • Shows points remaining/needed to win for each player
  • Automatic timer can be used to monitor time for each shot and time-out (up to 30 minutes)
  • Match timer tracks time for the match (up to 24hrs)
  • Remembers a match for up to 30 days

How to use:

  • Tap the player name to edit, and enter player name
  • Tap the points needed value to edit (the one to the right of the slash (/)
  • Tap to the left of the starting player to set that player's turn; the current player will be shown in bold with a cue ball displayed next to it.
  • Tap "Start Match"
  • Tap each ball as the player sinks it; tap again to mark as "dead".
  • To correct a ball, use repeated taps to cycle through each of player's ball, dead, opponent's ball, and still in-play.
  • Use End Turn to advance to the next player and automatically count innings. Use the innings +/- buttons to correct as required
  • Use Next Rack to advance to the next rack.

Controls and what they do:

  • Start Match
    Resets scoring to begin a match, records the start time, and starts the shot timer.
  • End Match
    Stops timers and posts the final rack score to the summary.
  • Clear Rack
    Resets the current rack to a state as if it was never played.
  • Summary
    Displays a summary of completed racks.
  • Innings +/-
    Adjusts the inning count for the current rack.
  • Defensives +/-
    Adjusts the defensives count. You can't remove defensives that weren't added in the current rack.
  • Balls 1-9
    Marks the ball and resets the shot timer.
  • End turn
    Advances to the next player, records the inning as appropriate, and restarts the shot timer.
  • Time-out
    Restarts the shot timer to monitor the duration of a time-out.
  • 9 Snap
    Records a 9 on the break and advances to the next rack. Make sure you mark any other balls first.
  • 9 Run
    Records a 9-and-run and advances to the next rack. All balls in the rack are awarded to the current player.
  • Next Rack
    Advances to the next rack. Note that you can only adjust scoring for the rack currently in progress.
  • Hourglass
    Restarts the shot timer. You can use this at any time.
